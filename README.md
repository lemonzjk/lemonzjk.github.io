# Jiankang Zheng Academic Homepage

This repository contains Jiankang Zheng's academic homepage, adapted from the open-source [WowPage](https://github.com/WD7ang/WowPage) template.

## Main Files

- `_pages/about.md`: homepage content, including news, experience, publications, projects, awards, and skills.
- `_config.yml`: site metadata and author profile.
- `_data/navigation.yml`: top navigation.
- `assets/css/home.css`: homepage-specific styling.
- `images/`: profile image and publication/project thumbnails.
- `files/cv-zh.pdf`: Chinese CV linked from the navigation bar.
- `DEPLOY.md`: GitHub Pages deployment guide.

## Deployment

This is a GitHub Pages Jekyll site. Push the repository to GitHub and enable Pages from `main / root`.

See [DEPLOY.md](DEPLOY.md) for details.

## Local Preview

```bash
./scripts/preview.sh
```

Then open:

```text
http://127.0.0.1:4000/
```
